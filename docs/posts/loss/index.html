1.pretrain阶段的loss#

\[ Input = <bos>, w_1, w_2...w_T \]

\[ Label=w_1,w_2...w_T,<eos> \]

\[ Loss_{pretrain} = \sum_{k=1}^{T+1} \log{P(w_t|<bos>, w_1, w_2...w_{t-1})} \]

\[ Label=\underbrace{[-100, -100...-100]}_{L_p+1个ignoreIndex},r_1,r2...r_{L_t} \]
